# Forge OS Project Status

Forge OS is a Relationship Intelligence Platform for managing people, interactions, tasks, and business context across multiple ventures.

## Current Stack

- Next.js 16
- App Router
- TypeScript
- Tailwind v4
- shadcn/ui
- GitHub Codespaces

## Current Features

- Dashboard
- Sidebar navigation
- Person model
- Interaction model
- Task model
- Service layer
- Inbox page
- People page
- Dynamic person profile pages
- Interaction timeline
- Tasks on person profiles
- Dashboard counts driven from services

## Product Direction

Forge is not a CRM.

Forge is a Relationship Intelligence Platform.

## Core Entities

- Person
- Interaction
- Task
- Product
- Workspace
- Organization
